🔑 Claude API キー

Anthropic Console（console.anthropic.com）で取得したAPIキーを入力してください。このキーはブラウザのローカルストレージに保存され、外部には送信されません。

✍️ 差出人情報

ここに入力した内容が礼状の末尾（差出人名の下）に挿入されます。Gmailの署名と同じ内容を登録しておくと便利です。

📊 Googleスプレッドシート連携（任意）

Googleスプレッドシートに記録する場合のみ入力してください。GASのセットアップ手順は別途ドキュメントを参照してください。

✉️ Gmailアドレス（任意）

「Gmailで開く」ボタンの送信元として使用します。

AndroidでGmailアプリが開かない場合に変更してください。「mailto:」はAndroidの設定でGmailをデフォルトメールアプリにしている場合に有効です。「intent://」はGmailアプリを直接指定します。

🗑️ データ管理

ローカルに保存された全ての名刺データを削除します。この操作は元に戻せません。
